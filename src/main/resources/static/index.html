CRUD Operation Endpoint Request Body / Path Parameters Description / Response
GET /worker/{id} Path Parameter:
  • id: Integer - Идентификатор работника, которого нужно получить
 Получить конкретного работника по его идентификатору ID.
Responses:
  • 200 OK - Работник найден.
  • 404 Not Found - Если такого работника нет.
GET /worker None Получить список до 5 последних работников.
Responses:
  • 200 OK - Список объектов Worker (до 5).
POST /worker/new Request Body: 
{
    "name": "string (max 50, not blank)",
    "surName": "string (max 50, not blank)",
    "dateOfBirth": "yyyy-MM-dd (past)",
    "address": "string (not blank)"
}
 Создать нового работника.
Responses:
  • 201 Created - Создан объект Worker.
PUT /worker/update/{id} Path Parameter:
  • id: (Integer) - Идентификатор работника для обновления.
Request Body: 
{
    "name": "string (max 50, not blank)",
    "surName": "string (max 50, not blank)",
    "dateOfBirth": "yyyy-MM-dd (past)",
    "address": "string (not blank)"
}
 Обновить существующего работника по идентификатору ID.
Responses:
  • 201 Created - Обновлен объект Worker
  • 404 Not Found - Если такого работника нет
DELETE /worker/delete/{id} Path Parameter:
  • id: Integer - Идентификатор работника, которого нужно удалить
 Удаление работника по идентификатору ID.
Responses:
  • 200 OK - Если работник был успешно удален
  • 404 Not Found - Если такого работника нет
